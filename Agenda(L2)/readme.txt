Camille Tura--Durand 

Commandes :

c : Ajout d'un contact
a : Ajout d'un rdv
s : Supprimer un rdv
f : Afficher un rdv
q : quitter le programme
o : Oui pour la sauvegarde
n : Non pour la sauvegarde 

Fonctionnement :

Une fois le programme lancé, il faut tout d'abord choisir si l'on veut charger une sauvegarde en rentrant la lettre entre crochets pour choisir oui ou non. Puis choisir une des lettres entre crochets pour savoir quoi faire puis suivre les consignes données dans le cadre. 
Il y a une erreur d'affichage lorsque nous entrons une date de rendez-vous après l'entrée de notre réponse le message d'erreur s'affiche avant de nous demander l'heure du rendez-vous, il y la même erreur lorsqu'on nous demande la durée du rendez-vous. Pour l'affichage d'un rendez-vous, la date, l'heure et la durée ne s'affiche pas avec les bonnes informations.
